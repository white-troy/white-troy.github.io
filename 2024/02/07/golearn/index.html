前言

经过一个星期的学习，结合c++和python的语法格式，对于go有了一个初步的了解，在此记录下来，方便后续的温习

参考链接：go语言十分钟入门教程 - 掘金 (juejin.cn) ；Go 语言之旅 (go-zh.org)

关键字

这是go语言中的一些关键字，与c++不同的是，go中没有while关键字，而是使用for来实现while的相关功能

数据类型

在go语言中，数据类型用于声明函数和变量，按类别总共有以下几种数据类型

变量

在go语言中，类型决定了变量内存的长度和存储格式，我们只能修改变量值，无法改变类型。不同于c++的变量声明格式，go语言在定义时将类型放在变量后面，同时如果显示的提供了初始值，则可以省略变量类型，编译器会自动推断，未提供初始值则会定义为0等初始值。在go中，可以通过var关键字定义变量

在函数中，可以通过简短赋值语句对变量进行定义以及初始化，但只能在函数内部中使用，且不能提供数据类型，编译器会根据赋予的初始值进行自动推断。函数外的每个语句都必须以关键字开始（var,func等）

常量

常量表示运行时不可修改的值，常量值必须是可确定的字符，字符串，数值或布尔值。可以指定常量类型，或者由编译器通过初始化推断。在go中，通过关键字“const”来定义常量，在go中，没有明确的枚举类型，但可以通过iota标识符来实现一组自增常量值来实现枚举类型

函数

使用func关键字来定义函数，可以指定参数和返回值，函数定义的语法是func 函数名(参数列表) 返回值类型 { 函数体 }

流程控制语句

for

在go中只有一个循环结构，即是for循环。for循环与C/C++的for结构类似，同样由初始化语句，条件表达式，后置语句组成，但不同的是，这里没有小括号，但有大括号

for是go中的”while”

在for循环中，不写初始化语句和后置语句，同时将分号去掉的时候，for就变成了while的功能

if

和for类似，if在go中也是不需要小括号，但需要大括号

此外，if也可以在条件表达式之前执行一个简单的语句，该语句声明的变量的作用域仅在if之内，同时可以在对应的else中使用

switch

switch是编写一连串if-else语句的简单方式，与其他语言的switch语法格式类似，但在go中，switch只会运行选定的case，而不会运行所有的case，也就是go在case语句中自带了一个break语句，除非以fallthrough语句结束，不然case分支会自动终止。此外，switch的case无需为常量，取值也不必为整数

当switch语句没有具体的表达式或变量作为条件时，这种形式的switch被称为开关语句，下方的case会依次执行判断，直到找到一个匹配的条件分支

defer

defer语句会将函数推迟到外层函数返回之后执行，推迟调用的函数会立即求值，但直到外层函数返回前都不会调用这个函数。相当于defer对应的这个函数被压入了栈，然后按照后入先出的顺序依次调用

数据

数组

go提供了数组类型的数据结构，数组是具有相同唯一类型的一组已编号且长度固定的数据项序列，这种类型可以是任意的原始类型例如整型、字符串或者自定义类型。

数组的声明表达式为 var a[n]T，表示声明一个名为’a’的有n个T类型的值的数组，数组的长度是其类型的一部分，因此数组不能改变大小，可以通过索引来访问数组中的元素

切片

切片(slice)本身不是动态数组或动态指针，只是它内部采用数组存储数据，当数组长度达到数组容量是，会进行动态扩容。

创建切片

切片可以用内建函数make()来创建，make函数会分配一个元素为零值的数组并返回一个引用了它的切片

切片通过两个下标来界定，即一个上界和一个下界，二者以冒号分割，切片[ ]T的声明表达式为a[low: high]T，表示一个类型T的切片，从low切到high，是一个左闭右开的区间

切片的属性

切片拥有长度和容量，切片的长度就是他所包含的元素个数，可以通过len(s)来获取；切片的容量是从他的第一个元素开始数，到其低层数组元素末尾的个数，可以通过cap(s)来获取。

切片的零值是nil，nil切片的长度和容量都为0且没有底层数组

添加元素

向切片追加元素可以使用append函数，func append(s [ ]T, vs…T)[ ]T,append的第一个参数s是一个元素类型为T的切片，其余类型为T的值将追加到该切片的末尾。

当s的底层数组太小，不足以容纳所有给定的值时，他就会分配一个更大的数组

Range

for 循环的 range 形式可遍历切片或映射。当使用 for 循环遍历切片时，每次迭代都会返回两个值。第一个值为当前元素的下标，第二个值为该下标所对应元素的一份副本。

在range中，可以将下标或值用_来忽略它

指针

go的指针保存了值的内存地址，*T是指向T类型的指针，其零值为nil。&操作符会生成一个指向其操作数的指针，与C/C++不同的是，go没有指针运算

结构体

结构体(struct)将多个不同类型命名字段(field)序列打包成一个复合类型。字段名必须唯一，可用”_”补位，支持使用自身指针类型成员。字段属性为基本数据类型。字段可以通过 ‘.’ 方法来调用

这与c++的 ‘类’ 类似，结构体struct可以类比为class，字段属性可以类比为类成员变量，方法可以类比为类成员方法。

结构体字段可以通过结构体指针来访问。如果我们有一个指向结构体的指针 p，那么可以通过 (*p).X 来访问其字段 X。不过这么写太啰嗦了，所以语言也允许我们使用隐式间接引用，直接写 p.X 就可以。

结构体文法通过直接列出字段的值来重新分配一个结构体，使用name: value语法可以仅列出部分字段，特殊的前缀&返回一个指向结构体的指针

map字典

字典是一个使用频率较高的数据结构，作为无序键值对集合，字典key值必须是支持相等运算符的数据类型，比如数字、字符串、指针、数组、结构体，以及对应接口类型。通过make函数或初始化表达语句来创建

map的文法与结构体类似，但必须有键名，若顶级类型只是一个类型名，可以在文法的元素中省略它。
