STL string类

std::string 类属于C++标准库的头文件< string >，它在标准命名空间 std 中定义。使用 std::string 类需要包含 < string > 头文件，并在代码中使用 std::string 类型。

char和std::string处理字符串的区别

在c++中，可以用char和char*来处理字符串，同样的，也可以用std::string来处理，他们之间的区别如下：

  1. char 是 C++ 中的基本字符类型，用于表示一个单个字符。在内存中，char 类型通常占用一个字节（8位），可以表示 ASCII 字符集中的字符，范围是 -128 到 127 或 0 到 255（根据有无符号性而定）。如’char a = ‘A’’,在处理长字符串时，得通过char staticString[strlength]来处理，这个长度是静态的，无法调整静态数组的长度
  2. char 是指向字符的指针类型，也称为 C 风格字符串（C-style string）或字符数组,在 C++ 中，char 通常用于表示以空字符 ‘\0’ 结尾的字符序列，即以空字符为字符串的结束标志。如’char str = “Hello”‘，可以通过 char dynamicString = new char [strlength]来获得一个动态分配的字符数组，但是如果要在运行阶段改变数组长度，则必须先释放内存，在重新分配内存
  3. std::string 是 C++ 标准库中提供的字符串类，是一种动态字符串类型，用于表示可变长度的字符串。std::string 封装了字符序列，并提供了丰富的成员函数来方便地进行字符串操作。它可以自动处理字符串的内存管理，不需要手动分配或释放内存。STL 字符串类 std::string 和 std::wstring 分别模拟了普通字符串和宽字符串

截短STL string

STL string 类提供了 erase()函数，具有以下用途。

  1. 在给定偏移位置和字符数时删除指定数目的字符。
    string sampleStr (“Hello String! Wake up to a beautiful day!”);

    sampleStr.erase (13, 28); // Hello String!

  2. 在给定指向字符的迭代器时删除该字符。
    sampleStr.erase (iCharS); // iterator points to a specific character
  3. 在给定由两个迭代器指定的范围时删除该范围内的字符。
    sampleStr.erase (sampleStr.begin (), sampleStr.end ()); // erase from begin to end
