std::list

std::list 是 C++ 标准模板库（STL）中的一个双向链表容器。它是一个动态大小的容器，可以在任意位置高效地插入或删除元素，而不像 std::vector 或 std::deque 那样需要移动其他元素。双向链表的主要优点是，插入和删除元素的速度快，且时间是固定的。在使用时要包含头文件< list >此外，从c++11开始，也可以使用单向链表std::forward_list。

std::list的特点如下：

  1. 使用双向链表结构来存储元素，每个节点包含一个元素以及指向前一个和后一个节点的指针。这使得在任意位置添加或删除元素非常高效，时间复杂度为 O(1)。
  2. std::list 可以根据需要动态调整大小，自动处理内存分配和释放,这与vector类似
  3. 由于 std::list 是一个链表结构，不能像数组一样使用下标直接访问元素。但可以使用迭代器来遍历链表，支持正向和反向迭代。

在 list 中间插入元素

std::list 的特点之一是，在其中间插入元素所需的时间是固定的，这项工作是由成员函数 insert()完
成的。,insert有三种版本：

  1. 单个元素插入：insert 函数接受的第 1 个参数是插入位置，第 2 个参数是要插入的值。该函数返回一个迭代器，它指向刚插入到 list 中的元素
  2. 多个元素插入：该函数的第 1 个参数是插入位置，最后一个参数是要插入的值，而第 2 个参数是要插入的元素个数。
  3. 范围插入：该重载版本是一个模板函数，除一个位置参数外，它还接受两个输入迭代器，指定要将集合中
    相应范围内的元素插入到 list 中。注意，输入类型 InputIterator 是一种模板参数化类型，因此
    可指定任何集合（数组、vector 或另一个 list）的边界

删除 list 中的元素

list 的成员函数 erase()有两种重载版本：一个接受一个迭代器参数并删除迭代器指向的元素（单个元素删除）；另一个接受两个迭代器参数并删除指定范围内的所有元素（范围元素删除）。

  1. 单个元素删除：
    删除指定位置 pos 处的单个元素，并返回指向被删除元素之后元素的迭代器。
  2. 范围元素删除：
    删除范围 [first, last) 内的所有元素，并返回指向被删除元素之后元素的迭代器。注意这里左闭右开，所以被删除的值不包括last指向的值
  3. 注解：
    1.注1：在单个元素删除版本内，为什么删除第三个元素是it = numbers.erase(it);而不是number=numbers.erase(it)？
    答：当调用 erase() 成员函数删除单个元素时，它会在删除元素后返回一个指向被删除元素之后元素的迭代器。由于在删除元素后链表的结构会发生改变，原先指向被删除元素的迭代器 it 就会变得无效，无法再用于遍历链表或其他操作。因此，必须使用 erase() 的返回值来更新迭代器 it，使其指向有效的位置。
    而而 number = numbers.erase(it); 是错误的写法，因为 number 的类型是 int，而 erase() 返回的是一个迭代器，将一个迭代器赋值给 int 类型是不合法的。
    2.注2：在范围删除时，使用 erase() 删除一系列元素后，迭代器 itStart 和 itEnd 的值不需要更新。这是因为在 erase() 的范围删除版本中，itStart 和 itEnd 分别指向要删除的范围的开始和结束位置，而 erase() 会在删除元素后自动更新这两个迭代器的位置，使它们仍然有效

对元素进行排序

std::list 提供了 sort() 成员函数，用于对链表中的元素进行排序。sort() 函数默认按照元素的自然顺序（升序）进行排序，要求元素的类型支持比较操作。如果元素的类型没有定义比较运算符，需要通过自定义比较函数来进行排序。

  1. 默认排序：这个版本会按照元素的自然顺序（升序）对链表中的元素进行排序。
  2. 自定义排序版本：template < class Compare > void sort(Compare comp);接受一个二元谓词函数作为参数，允许通过自定义比较函数 comp 来指定排序的规则。比较函数 comp 接受两个元素作为参数，并返回一个布尔值，用于指定元素的顺序关系。如：
