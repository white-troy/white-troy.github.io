动态数组vector

std::vector 是 C++ 标准模板库（STL）中的一个顺序容器，用于存储动态大小的元素序列。它提供了动态数组的功能，并且在内存中连续存储元素，因此支持高效的随机访问，使用时要包含头文件< vector >，它的特点如下：

  1. 在 vector 容器中，根据下标随机访问某个元素的时间是常数，在尾部添加一个元素的时间大多数情况下也是常数，即在末尾插入元素的所需时间不随数组大小而异。
  2. 在中间插入或删除元素时，因为要移动多个元素，因此速度较慢，所需的时间与该元素后面的元素个数成正比。
  3. 在 vector 容器中，用一个动态分配的数组来存放元素，因此根据下标访问某个元素的时间是固定的，与元素个数无关，存储的元素数是动态的，而 vector 类负责管理内存。

查询vector的大小和容量

在 C++ 的 std::vector 中，大小（size）和容量（capacity）用于描述向量中元素的数量和内部存储的容量。要查询 vector 当前存储的元素数，可调用 size()： integers.size ();
要查询 vector 的容量，可调用 capacity( )：integers.capacity ()

vector 的大小指的是实际存储的元素数，大小与元素的实际数量直接相关，即 std::vector 中存储的元素数量就是它的大小。而 vector 的容量指的是在重新分配内存以存储更多元素前 vector 能够存储的元素数，当向量的元素数量超过当前容量时，向量会重新分配更大的内存空间，以容纳更多元素。这个过程称为 “重新分配”（reallocation）。因此，vector 的大小小于或等于容量。

deque类
